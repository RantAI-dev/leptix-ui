# leptix-checkbox

Leptix Checkbox component — a control that allows toggling between checked, unchecked, and indeterminate states.

Part of [Leptix](https://github.com/RantAI-dev/leptix-ui) — Radix-quality accessible UI primitives for Leptos.

See the [main documentation](https://github.com/RantAI-dev/leptix-ui) for installation, usage examples, and the full component list.
