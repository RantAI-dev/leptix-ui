# leptix-context-menu

Leptix Context Menu component — a menu triggered by right-click, extending the Menu primitive.

Part of [Leptix](https://github.com/rantai-dev/leptix) — Radix-quality accessible UI primitives for Leptos.

See the [main documentation](https://github.com/rantai-dev/leptix) for installation, usage examples, and the full component list.
